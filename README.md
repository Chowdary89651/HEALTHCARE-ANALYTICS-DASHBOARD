# Healthcare Analytics Dashboard

## Project Overview
This project is an interactive Power BI dashboard that analyzes healthcare data to provide insights into patient admissions, hospital performance, treatment costs, diagnoses, and operational trends.

## Tools Used
- Power BI
- Microsoft Excel
- Power Query
- DAX

## Dashboard Features
- Total Patients
- Total Admissions
- Average Treatment Cost
- Average Length of Stay
- Department-wise Analysis
- Disease-wise Analysis
- Monthly Admission Trends
- Interactive Slicers and Filters
